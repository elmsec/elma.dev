[Telegramic](https://telegramic.org) is both a community for Telegram users and developers, and a Telegram directory containing bots, channels, groups, stickers, news, and so forth!

### Used technologies
* GNU/Linux
    * Ubuntu
* Python
    * Django
* PostgreSQL
* Celery
* Bash, shell script
* Fomantic UI (Semantic UI fork)
* Telegram API
    * Telethon
    * Pyrogram
* Telegram Bot API
    * python-telegram-bot
